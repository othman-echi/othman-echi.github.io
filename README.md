# Othman Echi Academic Homepage

This repository contains the academic homepage of **Professor Othman Echi**,
Professor of Mathematics at University of Tunis el-Manar, Tunisia.

The homepage presents research areas, selected publications, teaching,
academic service, contact information, and a downloadable CV.

## Website

When published with GitHub Pages, the site will be available at:

```text
https://othman-echi.github.io
```

## Main Files

```text
index.html
index.htm
assets/site.css
assets/site.js
assets/othman-echi-photo.jpg
assets/Othman_Echi_CV.pdf
```

## How to Update

- Edit `index.html` to update the homepage text, research sections, teaching,
  service, links, and contact information.
- Edit `assets/site.js` to update the publications list and publication filters.
- Edit `assets/site.css` to adjust the visual design.
- Replace `assets/Othman_Echi_CV.pdf` when a new CV is available.
- Replace `assets/othman-echi-photo.jpg` to update the homepage photo.

For GitHub Pages, `index.html` is the main entry file. The duplicate
`index.htm` is kept for compatibility with older hosting setups.

## Deployment

In the GitHub repository settings:

1. Go to **Settings**.
2. Open **Pages**.
3. Select **Deploy from a branch**.
4. Choose the `main` branch and `/ root`.

GitHub Pages will then publish the site automatically.
